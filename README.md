# beverage-preferences-dashboard

##About
This project analyzes the beverage preferences of consumers using a dataset I created in Excel, transformed using SQL, and visualized using Power BI.
The dashboard highlights trends in the average ratings of flavors, weekly purchases of flavors, sweetness levels, and demographic differences among tartness preference. It was designed to demonstrate an end-to-end workflow following data creation -> cleaning -> transformation -> visualization. 

##Tools
Excel - Created and structered the dataset
SQL - Filtered, grouped, and aggregated data
Power Bi - Built interactive dashboard and visualized insights

- beverage-preferences-dashboard.pbix — Power BI dashboard
